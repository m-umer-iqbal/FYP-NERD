# Frontend + Backend of NERD

## How to start the project
(In Terminal, run the following command)

1. Clone the repository

```bash
git clone https://github.com/m-umer-iqbal/FYP-NERD.git
```

2. Install dependencies for both frontend and backend 

```bash
cd backend
npm install
cd ../frontend
npm install
```

4. Start the frontend server

```bash
npm run dev
```

5. Open your browser and type this in address bar  `chrome://extensions/` to access the extensions page, then enable the developer mode from the `right top corner toggle button`, then click on `load unpacked` button in the left top corner and finally locate add the `dist folder`.

ALSO YOU CAN DO THIS (optional)

6. Open your browser and type this in address bar `http://localhost:5137` to access the extension frontend in full display.

## Project Structure
- `frontend/`: Contains the React + Tailwind frontend code.
- `backend/`: Contains the Node.js backend code.

## Technologies Used
- Frontend: React, Tailwind CSS
- Backend: Node.js, Express
